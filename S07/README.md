# S7 — Calculadora básica

Calculadora con operaciones fundamentales y pruebas unitarias con PHPUnit.

## Operaciones

- Sumar
- Restar
- Multiplicar
- Dividir (con validación de división por cero)

## Requisitos

- PHP 8.0+
- Composer

## Instalación

```bash
composer install
```

## Ejecutar pruebas

```bash
php vendor/bin/phpunit tests
```
